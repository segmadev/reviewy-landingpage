USER FLOW & LINKING LOGIC
JOB SEEKER: Login -> Upload -> Target Job -> Fill Info (Steps 2-7) -> Result.
BROADCAST: Result -> Dashboard -> Toggle "Pulse" -> Radar view.
RECRUITER: Login -> Select Pricing -> Browse Pulse Snippets -> Unlock CV.
SETTINGS: User clicks Avatar (Top-Right) -> Profile/Settings page.
RESPONSIVE (MOBILE) SPECIFICATIONS
Sidebar: Collapses to Hamburger menu (top-left).
Split-Screen: The Live Preview (Right) moves to a "Preview" tab or bottom.
Tables/Lists: Convert to vertically stacked cards.
Forms: All 2-column inputs become 1-column.
ASSET & TECHNICAL NOTES
White Space: AI should ignore fixed SVG export padding; use 'object-fit: contain'.
Icons: Use Lucide-React or Phosphor icons matching the SVG styles.
State: Builder must update CV Preview on every 'onChange' event.
